# 0. MySQL的安装 - Windows


### 0.MySQL的安装-Windows


### 1.本节⽬标

•掌握在Windows系统下使⽤Installer安装MySQL数据库
•掌握在Windows系统下安装MySQL历史版本
MySQL版本：8.0.x
安装之前先确保没有MySQL服务正在运⾏，建议在没有安装过MySQL的机器上进⾏全新的安装。

### 2.使⽤Installer安装


### 2.1下载最新版本Installer

•官⽹下载8.0.X的最新版本，⽹址： https://dev.mysql.com/downloads/installer/

### 8.0.X中的X是⼩版本号，每发布⼀个⼩版本就会累加⼀次，选最新即可。


下载的过程中需要登录，如果没有Oracle的账⼾，注册⼀个即可。

### 2.2运⾏安装程序


### 1.点击Edit并设置过滤器


### 2.确保所有组件都显⽰在左侧列表中



### 3.依次展开每个组件，选择要安装的版本，通常选最新的即可



### 4.点击向右的箭头，将选择的版本移⾄待安装区，依次选中组件出现⾼级选项

⼀般只选择MySQLServer和MySQLWorkbench这两个组件，其他的现在⽤不到，课件中为了完整演⽰安装过程选择了所有组件


### 5.设置⾃定义安装路径，建议安装在⾮系统盘，点击OK后，再点击Next



### 6.点击执⾏按钮



### 7.数据库安装完成，点击下⼀步



### 8.点击下⼀步配置MySQL服务器
