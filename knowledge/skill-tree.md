# 软件测试工程师技能树

> **视角**: Agent 测试实习生 → 资深测试工程师
> **短期目标**: 掌握公司流程化测试，胜任当前实习工作
> **长期目标**: 成为具备架构视野的资深测试工程师

---

## 阅读指南

- `[P0]` = 实习生阶段必须掌握（3个月内）
- `[P1]` = 初级工程师阶段（3~12个月）
- `[P2]` = 中级工程师阶段（1~3年）
- `[P3]` = 高级/资深工程师阶段（3年+）
- `→` 表示前置依赖，箭头左侧是右侧的前置技能
- `★` 表示对 Agent 测试方向特别重要的技能

│   ├── 斐波那契 → 爬楼梯（DP 启蒙）
│   ├── 选或不选模型（打家劫舍）

**练习方式**: 先讲后练——先学算法思路，再动手写代码，每日 3-4 题。
